# Portfolio de Proyectos

Repositorio destinado a documentar proyectos de automatización, análisis de datos y generación de indicadores desarrollados en entornos corporativos.

## Proyectos

- Charla-5-Min
- Capacitaciones
- Gestor de Consultas

Cada proyecto se presenta como un caso de estudio, describiendo el problema abordado, la solución implementada, las tecnologías utilizadas y los resultados obtenidos, sin divulgar información confidencial ni propiedad intelectual.
